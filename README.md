# PQ-PQ Goodness-of-Fit Criterion

Реализация и исследование мощности нестандартного критерия согласия, основанного на симметричном расстоянии между распределениями.

## 1. Теоретическая база

Статистика критерия измеряет квадратичное отклонение между композициями функций распределения и квантильных функций:

$$\rho_{PQ-PQ}(F, F_0) = \int_{0}^{1} \left( F(F_0^{-1}(x)) - F_0(F^{-1}(x)) \right)^2 dx$$

Где:
* $F$ — эмпирическая функция распределения (ECDF).
* $F_0$ — теоретическая функция распределения.
* $F^{-1}, F_0^{-1}$ — соответствующие квантильные функции.

В отличие от классического критерия Колмогорова-Смирнова, который ищет максимальное отклонение в одной точке, данный критерий интегрирует разности по всей области определения, учитывая как вероятностные (P-P), так и квантильные (Q-Q) расхождения.
